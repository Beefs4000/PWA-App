## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git@github.com:Beefs4000/PWA-App.git
   ```
2. Install NPM packages in the Root folder
   ```sh
   npm i
   ```
3. Once installed run the application 
   ```sh
   npm run start dev 
   ```
